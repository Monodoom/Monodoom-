Accesso al Database EEPROM

I parametri "P" (P-Settings) sono le costanti di base salvate nella memoria fisica del display Minimotors EY3. Questi valori determinano esattamente come la centralina interpreta il segnale dell'acceleratore magnetico e come gestisce il prelievo di Ampere dal pacco batteria. Un'errata configurazione può simulare un guasto hardware (es. taglio di potenza precoce, tachimetro sballato) o usurare irrimediabilmente le celle al Litio. Procedere con estrema cautela.

>> Log Dati Telemetrici (OSINT / Forense)

> Operazione: Accesso root all'interfaccia EY3.

> Comando Esecutivo: Tenere premuto il tasto [MODE] per 3 secondi. Premere singolarmente [MODE] per scorrere i parametri. Premere il tasto [DOT] (pallino) per modificare il valore selezionato.

> Warning Critico: Non alterare assolutamente i parametri P0, P1, P2 e P3. L'inserimento di valori di voltaggio errati manderà il BMS in blocco protettivo immediato.

Costanti di Configurazione

Parametro Funzione (Hardware) Setup Consigliato (Forense)
P0 Diametro Ruota (Pollici) 8.0 / 10.0 / 11.0 (In base al modello posseduto. Serve a calibrare il tachimetro).
P1 Voltaggio Sistema (Cut-off) 52V / 60V / 72V (Incrociare rigorosamente col voltaggio nominale della batteria).
P2 Poli Magnetici Motore 10 o 15 (Valore pre-impostato di fabbrica. Non alterare mai).
P3 Segnale Tachimetro 0 (Lettura dal sensore Hall interno al motore).
P4 Unità di Misura 0 (Km/h) / 1 (Mph).
P5 Zero Start (Modalità di Partenza) 0 (Parte da fermo - Pericoloso) / 1 (Richiede spinta col piede - Sicuro).
P6 Cruise Control (Pilota Automatico) 0 (Off) / 1 (On - Mantiene la velocità se il gas è fisso. Sconsigliato in città).
P7 Accelerazione Iniziale (Spunto) 0 (Erogazione Max - Scatto brutale) / 5 (Erogazione Morbida - Partenza dolce).
P8 Taglio Potenza (Limitatore %) 100 (Solo in aree private. Rilascia il 100% della corrente nominale ai motori).
P9 Battery Saver Mode (Erogazione Eco) 1 (Max risparmio energia) / 2 (Medio) / 3 (Nessun risparmio - Eroga tutta la corrente dispoibile finché la batteria non si esaurisce).
PA Frenata Elettromagnetica (E-ABS) 0 (Off) / 1-5 (Forza del freno motore. Usare 2 o 3 per non friggere i MOSFET della centralina rigenerando troppa corrente).
PB Intensità Retroilluminazione LCD 0 (Spento) / 5 (Max luminosità).
PD Auto Spegnimento ABS 0 (ABS Off) / 1 (ABS On - Crea micro-frenate a scatti per evitare il bloccaggio ruota. Spesso disattivato dai rider esperti).
